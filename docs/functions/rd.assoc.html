    Note

    This copies and flattens prototype properties onto the new object as well. All non-primitive properties are copied by reference.

    See also dissoc, pick.

    Example

    R.assoc('c', 3, {a: 1, b: 2}); //=> {a: 1, b: 2, c: 3}
